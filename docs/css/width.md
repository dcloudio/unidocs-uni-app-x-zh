#### App平台  
属性值为长度`<length>`时可以不设置单位，无单位时当做px处理。  

#### Web端
属性值为长度`<length>`时必须设置单位，无单位时当做非法值处理。  

> 实际项目中为了更好的在各端兼容，推荐指定明确单位。
