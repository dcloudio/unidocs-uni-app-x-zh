#### App平台  
App平台不支持指定多个时长。  
属性值可以不设置单位，无单位时当做ms处理。  

#### Web端
属性值必须设置单位，无单位时当做非法值处理。  

> 实际项目中为了更好的在各端兼容，推荐指定明确单位。
