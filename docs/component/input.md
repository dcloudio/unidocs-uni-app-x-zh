## Tips
- 从uni-app x 4.0起，App-Android平台 input 点击输入框外的屏幕会自动收起软键盘。
- 从uni-app x 4.0起，App-Android平台 input 的 font-size 默认值统一为 16px。
- web端iOS safari软键盘弹出时，整个页面会上推而不是挤压，从而导致导航栏可能会上移到屏幕之外。
