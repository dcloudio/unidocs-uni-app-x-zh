## databaseForJQL() @databaseforjql

**和uni-app项目接口差异**

- uni-app x项目内不再支持uniCloud.database()方法，仅支持uniCloud.databaseForJQL()。
- 不再支持action云函数，因为安全问题已经不再推荐使用。开发者应改用[数据库触发器](https://doc.dcloud.net.cn/uniCloud/jql-schema-ext.html)来实现相关功能。
- 暂不支持multiSend
