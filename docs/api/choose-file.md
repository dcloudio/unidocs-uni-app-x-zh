## tips
- chooseFile在App平台是系统UI，其风格不同rom可能有差异。多选时有的是长按、有的是checkbox。系统UI的暗黑模式、国际化跟随系统，而不跟随App。
- Android平台的chooseFile可以选择录音机，新录一段声音。此时可用于替代录音API `uni.getRecorderManager()`
