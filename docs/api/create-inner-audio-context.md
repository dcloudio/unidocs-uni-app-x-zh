## uni.createInnerAudioContext() @createinneraudiocontext

**注意：使用完后，必须调用destory方法将资源进行释放**

**支持格式**

|格式|web|Android|
|:-|:-|:-|
|flac	|√|√|
|m4a	|√|√|
|ogg	|√|√|
|ape	|√|√|
|amr	|√|√|
|wma	|√|√|
|wav	|√|√|
|mp3	|√|√|
|mp4	|√|√|
|aac	|√|√|
|aiff	|√|x|
|caf	|√|x|

**缓存说明**

播放的网络音频，默认会缓存到应用cache目录的uni-audio文件夹下，默认大小为100M,超过后会根据最近最少使用的缓存算法自动进行清除
